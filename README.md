# Sistema de Control de Bibliotecas (Código Base)

Este repositorio contiene un Sistema de Control de Bibliotecas implementado en
Python usando Programación Orientada a Objetos (POO).

⚠️ IMPORTANTE:
- El código **NO cumple el estándar PEP-8**.
- Los nombres de clases, variables y métodos **no son representativos**.
- El objetivo del estudiante es **refactorizar el código**, aplicando PEP-8,
  mejores prácticas de legibilidad y nombres semánticos.

## Actividad del estudiante
1. Realizar un fork del repositorio.
2. Analizar el código usando linters y revisión manual.
3. Corregir los problemas de estilo PEP-8.
4. Mantener la funcionalidad original.
5. Subir los cambios al repositorio forked.

## Restricciones
- Lenguaje: Python 3
- Paradigma: Programación Orientada a Objetos
